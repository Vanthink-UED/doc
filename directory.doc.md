## 项目目录说明

```javascript

+ application // 程序目录
+ conf
|   + application.ini  yaf配置文件
+ config  // 存放服务器配置文件 nginx or apache 配置
+ logs // 存放日志
+ projects  // 存放shell 脚本
+ public // 存放前端资源
|    + img  // 图片资源
|    + js // javascript 资源
|        + lib 常用的库
|        + directive  // angular.js diretive
|        + controller  // angular.js controller
|        + service  // angular.js service
|    + css  //css 资源
|    + less  //less
|    + fonts // 字体
|    + resource // 其他资源  mp3 mp4
|    + index.php yaf 入口框架
|    +  gulpfile.js gulp 文件
.htaccess apache rewrite



```
